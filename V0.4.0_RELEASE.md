## ğŸ› Known Issues & Workarounds

### Windows
**Issue:** Battery shows 0% or Unknown  
**Solution:** Ensure WMIC is available: `wmic /?`

**Issue:** Process kill requires admin  
**Solution:** Run as Administrator for system processes

### Linux
**Issue:** Battery not detected  
**Solution:** Check `/sys/class/power_supply/` directory (code checks BAT0 and BAT1)

**Issue:** Permission denied killing process  
**Solution:** Use `sudo` or check process ownership

### macOS
**Issue:** Battery health always 100%  
**Note:** macOS only reports Normal/Replace Soon (expected behavior)

**Issue:** Some processes require sudo  
**Solution:** Run with `sudo` for system processes
