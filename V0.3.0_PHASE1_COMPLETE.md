### 2. Export Data to CSV/JSON ðŸ“Š

**Keyboard Shortcuts:**
- `E` - Export to JSON (default)
- `Ctrl+C` - Export to CSV

**What Gets Exported:**
- System information (device name, OS, uptime)
- CPU stats (model, cores, usage, frequency)
- Memory stats (total, used, available, usage %)
- Disk stats (total, used, available, usage %)
- Disk partitions (all mounted drives)
- Network stats (RX/TX totals and rates)
- Battery status (if available)
- Process list (all current processes with details)

**Export Formats:**

**JSON Format:**
```json
{
  "timestamp": "2025-12-07 15:30:45",
  "system": {
    "device": "NOPALLL",
    "os": "Windows 11 (26100)",
    "uptime_seconds": 28845
  },
  "cpu": {
    "model": "AMD Ryzen 5 3500U",
    "physical_cores": 4,
    "logical_cores": 8,
    "usage_percent": 30.5,
    "frequency_mhz": 2100
  },
  "memory": { ... },
  "processes": [ ... ]
}
```

**CSV Format:**
```csv
Systrix System Monitor Export
Timestamp,2025-12-07 15:30:45

=== SYSTEM INFORMATION ===
Device,NOPALLL
OS,Windows 11 (26100)
Uptime (seconds),28845

=== CPU ===
Model,AMD Ryzen 5 3500U
...
```

**File Naming:**
- Auto-generated: `systrix_export_YYYYMMDD_HHMMSS.json`
- Saved in current directory

## âš ï¸ Known Limitations

### Process Suspend/Resume on Windows
- Windows doesn't natively support SIGSTOP/SIGCONT
- PowerShell method may not work for all processes
- System processes cannot be suspended
- Requires additional tools (PSTools) for full support

**Workaround:**
- Use Process Explorer from Sysinternals
- Or install PSTools: `choco install pstools`

### Export
- Exports current snapshot only (no history yet)
- Files saved in current directory only
- No custom filename option in TUI (yet)
